## :watch: RODANDO A APLICAÇÃO

```
# Clone este repositório
$ git clone https://github.com/EduardoMoreira26/Shopping-Cart-ReactJS.git

# Execute o projeto em um editor de código ex: VsCode

# Instale as dependências pelo terminal.
$ yarn ou npm install 

# É necessário abrir dois terminais!
$ em um terminal rode o comando yarn server   //PARA INICIAR O ARQUIVO server.json (API)
$ no outro terminal rode o comando yarn start   //PARA INICIAR O PROJETO
```
